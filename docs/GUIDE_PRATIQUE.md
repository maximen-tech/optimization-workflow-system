# Guide Pratique - Utilisation du Système

## Comment utiliser le workflow d'automatisation

1. Choisissez votre type de tâche: [CODE] [DEBUG] [TEST] [DEPLOY] [REFACTOR] [RESEARCH]
2. Envoyez votre demande avec le pattern approprié
3. Le système exécute automatiquement le workflow complet

## Exemples d'utilisation

### Créer une nouvelle feature
\`@workflow CODE Crée endpoint POST /users avec tests\`

### Déboguer un problème
\`@workflow DEBUG Le test échoue avec erreur CORS\`

### Ajouter des tests
\`@workflow TEST Couvrir le module authentication\`

### Déployer
\`@workflow DEPLOY Prépare version 1.0.0 en production\`

### Refactoriser
\`@workflow REFACTOR Optimise la structure du bundle webpack\`

### Rechercher
\`@workflow RESEARCH Analyse des patterns React modernes\`
