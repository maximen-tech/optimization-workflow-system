# Quand Utiliser Quel Pattern

## Decision Tree

### Tu veux créer quelque chose ?
→ Utilise **[CODE]**

### Il y a un bug ?
→ Utilise **[DEBUG]**

### Tu veux tester du code ?
→ Utilise **[TEST]**

### C'est prêt pour production ?
→ Utilise **[DEPLOY]**

### Le code doit être optimisé ?
→ Utilise **[REFACTOR]**

### Tu as une question technique ?
→ Utilise **[RESEARCH]**
